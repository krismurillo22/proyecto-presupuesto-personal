# README

## Sistema de Presupuesto Mensual Personal

### Descripción

El Sistema de Presupuesto Mensual Personal es una aplicación web creada
para ayudar a las personas a organizar y controlar su dinero de forma
más consciente.
Permite registrar ingresos, gastos, metas de ahorro y presupuestos
mensuales, además de generar alertas automáticas cuando se alcanzan
ciertos límites o se cumplen metas.

El proyecto fue desarrollado para la clase de Teoría de Bases de Datos
I, aplicando los conceptos de diseño de bases de datos, normalización y
automatización mediante triggers, procedimientos almacenados y
funciones.

### Objetivos

**Objetivo general:**
Diseñar e implementar un sistema de presupuesto personal que permita al
usuario planificar, controlar y analizar sus finanzas mensuales.

**Objetivos específicos:**
- Crear un modelo de datos normalizado que refleje las entidades del
sistema financiero personal.
- Implementar procedimientos almacenados y triggers para automatizar
reglas de negocio.
- Desarrollar una interfaz web funcional para el manejo de presupuestos,
transacciones y metas de ahorro.

### Características principales

-   Registro de usuarios, presupuestos, transacciones y metas de
    ahorro.
-   Clasificación de ingresos y gastos por categorías y subcategorías.
-   Alertas automáticas cuando se superan límites de gasto o se cumple
    una meta.
-   Gráficos y reportes que muestran el comportamiento financiero.
-   Base de datos relacional estructurada y normalizada.

### Reglas de negocio principales

-   Un usuario solo puede tener un presupuesto activo por período.
-   Las transacciones deben estar asociadas a una subcategoría y
    coincidir con su tipo.
-   Las metas de ahorro solo se relacionan con subcategorías de tipo
    ahorro.
-   Las alertas se generan automáticamente según las condiciones del
    presupuesto o las metas.
